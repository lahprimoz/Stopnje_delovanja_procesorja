# Stopnje_delovanja_procesorja
Vse datoteke za izvedbo dejavnosti za prikaz stopenj delovanja procesorja (tj. fetch-decode-execute cikel).

Dejavnost je namenjena za izvedbo v zadnjem triletju osnovne šole in prikazuje delovanje centralne procesne enote v računalniku.
Z manjšimi spremembami se lahko dejavnost prilagodi za mlajše ali starejše učence.
S stopnjevanjem lahko postopno povečamo težavnost in hkrati dejavnost postane bližje realnemu delovanju modernega procesorja.
Z uvedbo hkratnega izvajanja korakov pa lahko uvedemo tudi idejo cevovoda (pipeline).

Glavna datoteka repozitorija je **Stopnje delovanja procesorja (dejavnost).pdf**, ki vsebuje vsa osnovna navodila za izvedbo dejavnosti.  
Dodani sta tudi dve natisljivi datoteki, ki predstavljata pomnilnik in se lahko uporabita pri izvedbi.
Ena od njih je neizpolnjena tabela, v katero se lahko vpiše program, ki ga bo hipotetična CPE izvedla.
Druga pa je podobna tabela, ki pa že vsebuje vpisan program za števec.
